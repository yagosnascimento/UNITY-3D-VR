
# VR Player Setup - Unity 6 (XR Interaction Toolkit)

Este repositório contém a configuração inicial de um Player VR utilizando o **Unity 6** e o **XR Interaction Toolkit**. O objetivo deste projeto é documentar o processo de setup de movimentação das mãos e resolução de problemas comuns na integração com o Input System.

## Tecnologias Utilizadas
* **Unity 6** (Versão 6000.0.6f1)
* **XR Interaction Toolkit (XRI)**
* **OpenXR Plugin**
* **Unity Input System**

## Configuração do Player
O Player foi estruturado seguindo o padrão `XR Origin (VR)`:
1. **Camera Offset:** Garante o rastreamento da altura correta do HMD.
2. **Main Camera:** Atua como o centro da visão do usuário.
3. **Left/Right Hand Controller:** Objetos responsáveis por rastrear a posição e rotação dos controles.
